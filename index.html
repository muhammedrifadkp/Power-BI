üìö Theory: Introduction to Power BI

What is Business Intelligence (BI)?

Business Intelligence is the process of collecting, analyzing, and presenting business data to help organizations make informed decisions.

Traditional Challenges:

  • Data scattered across multiple systems (Excel, databases, cloud services)
  • Manual report creation taking hours or days
  • Difficulty visualizing trends and patterns
  • Limited collaboration and sharing capabilities
  • No real-time insights

What is Microsoft Power BI?

Power BI is a business analytics service by Microsoft that provides interactive visualizations and business intelligence capabilities with an interface simple enough for end users to create their own reports and dashboards.

Key Capabilities:

  • üìä Data Connection: Connect to 100+ data sources
  • üîÑ Data Transformation: Clean and shape data easily
  • üßÆ Data Modeling: Create relationships and calculations
  • üìà Visualization: Build interactive reports and dashboards
  • üåê Sharing: Publish and collaborate in the cloud
  • üì± Mobile Access: View reports on any device

Power BI Components: The Ecosystem

Power BI
Power BI
Desktop
The Windows-desktop-based application for PCs and desktops, primarily for designing and publishing reports to the Service.
Power BI
Service
The SaaS (software as a service) based online service (formerly known as Power BI for Office 365, now referred to as PowerBI.com or simply Power BI).
Power BI
Mobile Apps
The Power BI Mobile apps for Android and iOS devices, as well as for Windows phones and tablets.
Power BI
Gateway
Gateways used to sync external data in and out of Power BI. In Enterprise mode, can also be used by Flows and PowerApps in Office 365.
Power BI
Embedded
Power BI REST API can be used to build dashboards and reports into the custom applications that serves Power BI users, as well as non-Power BI users.
Power BI Report
Server
An On-Premises Power BI Reporting solution for companies that won't or can't store data in the cloud-based Power BI Service.
Power BI Visuals
Marketplace
A marketplace of custom visuals and R-powered visuals.

üíª Installation & Setup

System Requirements

Minimum Requirements:
  • OS: Windows 10 or later (64-bit)
  • Processor: 1 GHz or faster
  • RAM: 2 GB minimum (4 GB+ recommended)
  • Display: 1440 x 900 resolution
  • Internet: Required for download and updates
  • Storage: 2-3 GB free space
üìù Important: Power BI Desktop is only available for Windows. Mac users need to use Windows virtual machine or Boot Camp.

Installation Methods

1

Method 1: Microsoft Store (Recommended)

Advantages: Automatic updates, easier installation

  1. Open Microsoft Store on Windows
  2. Search for "Power BI Desktop"
  3. Click "Get" or "Install"
  4. Wait for installation to complete
  5. Launch Power BI Desktop from Start Menu
2

Method 2: Download Center

Advantages: Offline installer, better for corporate environments

  1. Visit: https://powerbi.microsoft.com/desktop
  2. Click "Download Free"
  3. Run the downloaded PBIDesktopSetup_x64.exe file
  4. Accept license terms
  5. Choose installation location
  6. Click "Install"
  7. Launch after installation completes

First Launch Configuration

When you first open Power BI Desktop:

  1. Sign In (Optional): You can sign in with a Microsoft account or skip this step. Signing in allows you to publish to Power BI Service later.
  2. Welcome Screen: You'll see options to:
    • Get Data
    • Recent Sources
    • Open Other Reports
    • Sample Reports
  3. Close Welcome Screen: You can disable "Show this screen on startup"

Recommended Settings

‚öôÔ∏è

Adjusting Power BI Settings

Path: File ‚Üí Options and Settings ‚Üí Options

Global Settings:

  • Data Load: Disable "Autodetect new relationships after data is loaded" (for better control)
  • Power Query Editor: Enable "Show the Power Query Editor window when loading data"
  • Regional Settings: Set appropriate date format and language

Current File Settings:

  • Auto Save: Enable if working with OneDrive/SharePoint
  • Data Load: Review time intelligence settings

2. The Three Views

üìä Report View

Purpose: Create and design visualizations

  • Build charts and visuals
  • Arrange report layout
  • Add filters and slicers
  • Format and style
  • Most used view

üìã Table View

Purpose: View and validate data

  • See data in table format
  • Create calculated columns
  • Check data types
  • Sort and filter data
  • Verify transformations

üîó Model View

Purpose: Manage table relationships

  • View data model diagram
  • Create relationships
  • Edit relationships
  • Organize tables
  • View cardinality

3. The Right-Side Panes

üéØ Filters Pane

Apply filters at different levels:

  • Visual level: Filter specific visual
  • Page level: Filter entire page
  • Report level: Filter all pages

üìä Visualizations Pane

Contains two sections:

  • Visual Gallery: Select chart types (bar, line, pie, etc.)
  • Visual Properties: Format selected visual (colors, labels, legends)
  • Data Wells: Drag data to create visuals (X-axis, Y-axis, Values)

üìÅ Data Pane

Shows all available data:

  • List of all tables and columns
  • Calculated columns and measures
  • Hierarchies
  • Search functionality

Navigation Tips

üí°

Quick Navigation Shortcuts

  • Ctrl + S - Save report
  • Ctrl + Z - Undo
  • Ctrl + Y - Redo
  • Ctrl + C / Ctrl + V - Copy/Paste visuals
  • Ctrl + D - Duplicate visual
  • Alt + F4 - Close Power BI Desktop
  • F5 - Refresh data

üî® Practical Lab: Hands-On Exercises

‚è±Ô∏è Lab Duration: 90 minutes
üéØ Objective: Get comfortable with Power BI Desktop and create your first report

Exercise 1: Installation and Configuration (20 minutes)

1

Install Power BI Desktop

  1. Open Microsoft Store or visit powerbi.microsoft.com/desktop
  2. Download and install Power BI Desktop
  3. Launch the application
  4. Sign in with your Microsoft account (or skip)
  5. Close the welcome screen

‚úÖ Success Criteria: Power BI Desktop opens successfully with a blank canvas

2

Configure Settings

  1. Go to File ‚Üí Options and Settings ‚Üí Options
  2. Under Global ‚Üí Data Load:
    • Uncheck "Autodetect new relationships after data is loaded"
  3. Under Global ‚Üí Power Query Editor:
    • Check "Show the Power Query Editor window when loading data"
  4. Under Global ‚Üí Regional Settings:
    • Set locale to your region
  5. Click OK to save

Exercise 2: Interface Exploration (15 minutes)

3

Navigate the Interface

Task: Explore each component of the interface

  1. Ribbon: Click through each tab (Home, Insert, Modeling, View, Help)
  2. Views: Click the three view icons on the left side:
    • Report View (chart icon)
    • Table View (table icon)
    • Model View (relationship icon)
  3. Right Panes: Notice the three panes:
    • Filters (top)
    • Visualizations (middle)
    • Data (bottom)
  4. Try hiding/showing panes using the View tab

Exercise 3: Load Sample Data (15 minutes)

4

Import Your First Dataset

  1. Download the Sales_Sample.csv file (click button in Resources section)
  2. In Power BI Desktop, click Home ‚Üí Get Data ‚Üí Text/CSV
  3. Browse and select the Sales_Sample.csv file
  4. Click Open
  5. In the preview window, review the data
  6. Click Load (not Transform Data)
  7. Wait for the data to load

‚úÖ Success Criteria: You should see "Sales_Sample" table appear in the Data pane

5

Explore the Data

  1. Click on Table View (table icon on left)
  2. You should see the sales data in a table format
  3. Observe the columns: Date, Product, Category, Sales, Quantity, Region
  4. Note the data type icons next to each column name
  5. Click on different columns to sort the data
  6. Return to Report View

Exercise 4: Create Your First Visual (25 minutes)

6

Build a Simple Bar Chart

  1. Ensure you're in Report View
  2. In the Visualizations pane, click the Clustered Bar Chart icon
  3. An empty visual will appear on the canvas
  4. From the Data pane, drag:
    • Category to the Y-axis field well
    • Sales to the X-axis field well
  5. Your first chart is created! It shows sales by category
  6. Resize the visual by dragging corners
7

Add a Second Visual - Pie Chart

  1. Click on an empty area of the canvas (to deselect the bar chart)
  2. In the Visualizations pane, click the Pie Chart icon
  3. From the Data pane, drag:
    • Region to Legend
    • Sales to Values
  4. Position the pie chart next to your bar chart
  5. Resize to make it look balanced
8

Add a Card Visual (KPI)

  1. Click on an empty area
  2. Click the Card icon in Visualizations pane
  3. Drag Sales to the Data well
  4. This shows total sales
  5. Position it at the top of your report
  6. Make it wider and shorter (like a banner)

Exercise 5: Basic Formatting (15 minutes)

9

Format Your Visuals

  1. Select the Card visual (Total Sales)
  2. In Visualizations pane, click the Format (paint roller) icon
  3. Expand Callout value:
    • Change font size to 48
    • Change color to blue
  4. Expand General ‚Üí Effects:
    • Add a background color
    • Adjust transparency
  5. Select the Bar Chart
  6. Format it with:
    • Custom colors
    • Data labels (show values on bars)
    • Title (rename to "Sales by Category")

Exercise 6: Save Your Work

10

Save Your First Report

  1. Click File ‚Üí Save (or press Ctrl + S)
  2. Choose a location (Desktop or Documents)
  3. Name it: "Day1_MyFirstReport"
  4. Click Save
